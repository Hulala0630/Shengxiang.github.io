Unity C# OPC UA Simulation

A real-time visualization concept linking simulated machine states to a 3D Unity environment for operator review and engineering validation.

  • Problem: Machine behavior was difficult to explain before physical commissioning.

PLC Error Simulation System

PLC OPC UA Diagnostics Testing

A simulation setup for injecting realistic PLC faults and abnormal signal conditions to test alarm logic, error handling, and operator response.

  • Problem: Fault scenarios were hard to reproduce consistently during testing.
  • Solution: Created repeatable error states through controlled simulation inputs.
  • Result: Enabled earlier debugging of alarms, interlocks, and recovery sequences.
