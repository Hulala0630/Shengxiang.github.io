## 🎯 Focus Areas | 能力方向

### System Thinking | 系统思维  
Structuring ambiguous engineering problems into clear, testable systems.  
将模糊的工程需求转化为清晰、可验证的系统结构。

### Simulation & Validation | 仿真与验证  
Using simulation as a tool for early verification rather than late-stage debugging.  
将仿真作为前期验证手段，而非后期调试补救。  

---

### Data & Integration | 数据与集成  
Designing clear data flows across control systems, backend services, and visualization layers.  
在控制系统、软件服务与可视化之间构建清晰的数据流路径。  

---

### Software & Engineering Tools | 软件与工程工具  
Building modular tools to support engineering delivery, reuse, and scalability.  
构建模块化工程工具以支持交付、复用与扩展能力。  

---

### User Experience & Interaction Design | 用户体验与交互设计  
Making complex systems understandable and operable through clear interaction design.  
通过清晰的交互设计，让复杂系统具备可理解性与可操作性。
